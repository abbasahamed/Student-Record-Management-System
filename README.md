# Student Record Management System (C++)
This is a console-based application developed using C++ to manage student records.  
The program allows users to add, display, and search student information using object-oriented programming concepts.

## Features
- Add new student records
- Display all student records
- Search student by roll number
- Menu-driven console interface

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## Concepts Practiced
- Classes and Objects
- Arrays of Objects
- Functions
- Conditional statements
- Loops
